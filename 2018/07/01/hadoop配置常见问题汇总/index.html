2. jps显示正确，但是浏览器http://hadoop:50070却打不开

防火墙问题

  1. 查看防火墙是否打开
    sudo firewall-cmd --state
    显示

  2. 关闭防火墙
    sudo systemctl stop firewalld.service

  3. 检查是否关闭
    sudo firewall-cmd --state
    显示

  4. 重新用浏览器访问

  5. 禁止防火墙开机启动
    systemctl disable firewalld.service
